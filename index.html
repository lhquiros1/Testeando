Modelo simplificado a 10.000 Caras

Representación simplificada a 10.000 caras (proveniente de malla de de 668.414 caras).

Este modelo está optimizado para la publicación web, utilizando el formato estándar X3D para visores interactivos.

Pulse la tecla **'A'** para **centrar** rápidamente la escena y restaurar el punto de vista inicial predefinido.
